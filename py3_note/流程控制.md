1. 分支语句:
	1. 格式: if ... :
	         elif ... :
                 else ... :
	2. 特点:
		1. 可嵌套
		2. 同一代码块放入同一缩进
2. 循环语句
	1. while ... :
		可在下方加 else :
	2. for ... in ...:
		可以遍历任何可迭代对象
	3. range()
		1. 格式: range(终点)/range(起点/终点)/range(起点/终点/步长)
			*步长可为负
	4. break    退出当前循环
	5. continue 进入下一次循环
	6. pass     解释器跳过此句(用于占位)
